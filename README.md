# NHK番組表アプリ
NNHKのWeb Apiから番組情報を取得し、実際の番組表を表示するアプリケーションです。
# 機能一覧
・番組表表示
・番組詳細表示
# 使用した技術
フレームワーク・・・Spring Boot <br.
バックエンド・・・Java、My SQL My Batis<br> 
フロントエンド・・HTML、Bootstrap <br>
# 本アプリケーションを実行するにあたって
実行の前に必ず以下のSQLファイルからマスターデータを追加してください。<br>
・schema.sql・・・src/main/resources/schema.sql<br>
・data.sql・・・src/main/resources/data.sql<br> 

# 工夫した点 
・シンプルで見やすい見た目を意識しました。 <br>
・情報取得に必要なパラメータをデータベースにまとめることで、ユーザーが容易にパラメータを切り替えられるようになっています。 <br>

 

# 苦労したこと 
・Web Apiの仕組みを理解するのに苦労しましたが、疑問点をAIに問い合わせることで効率よく開発できるようになりました。 <br>
・また、エラーが発生した場合にもAIを活用することで問題個所の仕組みを理解しながら解決できるようになりました。 <br>
